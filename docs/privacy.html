Privacy Policy

Last updated: July 22, 2026

ChatakBoy is an automated Instagram messaging application operated by Ansh. It processes messages sent to the connected Instagram professional account so it can generate and send automated replies.

Information processed

The application may process Instagram-scoped user identifiers, message text, timestamps, and technical webhook data supplied by Meta.

How information is used

This information is used only to understand incoming messages, generate replies, prevent duplicate processing, operate the service, and troubleshoot errors.

Storage

Conversation context may be held temporarily in server memory. It can be cleared when the service restarts. The application does not sell personal information.

Third-party services

The application uses Meta's Instagram APIs, a hosted server provider, and an AI text-generation provider to operate.

Data deletion

To request deletion of data associated with your Instagram messages, send “delete my data” to the connected Instagram account or follow the data deletion instructions.

Contact

For privacy questions, contact the operator through the connected Instagram account.
